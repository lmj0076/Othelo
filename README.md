## OtheloGame

 자신이 놓을 돌과 자신의 돌 사이에 상대편 돌만 있는 경우에 돌을 놓을 수 있으며, 사이의 상대편 돌은 자신의 돌이 된다.

흑/백이 순서대로 한 번씩 돌을 놓는다.
놓을 곳이 없는 경우에는 다시 상대편 순서로 넘어간다.
